# 新闻稿生成器

这是一个面向校园广播场景的自动稿件生成项目，主要用于辅助广播员从学校官网、新闻网页或公告内容中整理信息，并生成适合播音的中文新闻稿、主持稿和栏目稿。

## 项目背景

本人是学校广播员，日常需要根据不同栏目准备播音稿。广播内容包括校园新闻、学校官网通知、综艺类栏目以及中午固定新闻稿等。为了提高整理稿件的效率，本项目尝试开发一个自动化工具，帮助完成新闻内容提取、稿件改写和打印稿生成。

## 计划功能

- 输入新闻链接或正文内容
- 自动提取标题、时间和正文
- 生成中文新闻播音稿
- 生成主持人口播稿
- 支持不同栏目模板
- 导出可打印的 Word 文档
